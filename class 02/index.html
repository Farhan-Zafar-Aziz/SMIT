Algebric Farmulas

    (a+b) 2 = a 2 + b 2 + 2ab

    (a-b) 2 = a 2 + b 2 - 2ab

    a 2 - b 2 = (a + b) (a - b)

    a 2 + b 2 = (a + b) 2 - 2ab

    a 2 + b 2 = (a - b) 2 - 2ab

    (a + b) 2 - (a -b) 2 = 4ab

    a 3 - b 3 = (a - b) (a 2 + ab + b 2 ) = (a - b) 3 + 3ab (a - b)

    a 3 + b 3 = (a + b) (a 2 - ab + b 2 ) = (a - b) 3 - 3ab (a - b)

    a 4 + b 4 = (a + b) (a - b) [(a + b) 2 - 2ab]

    a 3 + b 3 + c 3 - 3abc = (a + b + c) (a 2 + b 2 + c 2 - ab - bc - ca)
